Automation Anywhere Bot

This bot automates data patrol monitoring by extracting sysids from the ServiceNow UI and database, comparing records, and generating a report in Excel with missing entries. The bot also sends the report via email. This solution streamlines manual auditing and ensures timely reporting for integration gaps.

Tech Stack

  • Automation Anywhere A360
  • Excel automation package
  • Email automation
  • Database queries (ODBC)
  • ServiceNow UI scraping

Highlights

  • Fully unattended execution on scheduled basis
  • Excel summary with color-coded missing sysids
  • Error handling and logging for debugging
  • Reduces manual validation time from 4 hours to 10 minutes
