Manual Testing for SaaS Applications

Role: Tester
Tools Used: Azure DevOps, Sharepoint, ServiceNow

  • Overview: Conducted end-to-end manual testing for ServiceNow applications, ensuring system stability and seamless functionality. Designed and executed test cases, test plans, and test scripts based on business requirements. Identified defects, logged issues in Azure DevOps, and collaborated with developers for resolution.
  • Types of Testing Performed
    • Functional Testing: Verified core ServiceNow functionalities and workflows.
    • Regression Testing: Ensured new updates didn't break existing features.
    • Smoke Testing: Validated stability after deployments.
    • UI/UX Testing: Assessed user interface consistency and usability.
    • Integration Testing: Checked system interactions with third-party tools.

This project ensured a high-quality ServiceNow environment, reducing downtime and enhancing user experience.
