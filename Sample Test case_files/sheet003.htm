S1 S1.1 Validate WBS via Normal Travel Process TC1.1 Enter an invalid WBS number without a cost collector tied to it.

 The system does not accept the code. The message "Please enter a valid code" is displayed.  
TC1.2 Enter a valid WBS number without a cost collector tied to it. The system accepts the code and displays the message "You have a valid code". In the Main Details tab, related WBS fields (WBS Element, Company Code, Profit Center, Country/Location Associated to WBS & Cost Collector) are autopopulated.
TC1.3 Enter an invalid cost collector number. Cost Collector is not accepted by the system and message "Please enter a valid code" is displayed.
TC1.4 Enter a valid cost collector Cost Collector is accepted by the system and displays a message "You have a valid code". In the Main Details tab, related WBS fields (Charge Code, WBS Element, Company Code, Profit Center, Country/Location Associated to WBS) are populated with their corresponding values.
